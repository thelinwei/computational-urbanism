# Research notes and archive

This folder holds conceptual notes that sit underneath the pilots but aren't
themselves pilot outputs — the vocabulary discipline, the method-level
reasoning, and pointers into the broader Polymathic Learner archive.

- **[glossary.md](./glossary.md)** — the working-vs-formal language table. Which
  terms are used in formal proposals, and which stay private generative
  shorthand (and why the distinction matters).
- **[method-notes.md](./method-notes.md)** — the four-tier research logic
  (visibility → affordance → transmissibility → holding), what each tier can
  and cannot claim, and how the tiers depend on each other.

For the pilots themselves — the actual notebooks, data, and figures — see
[`../pilots/covered-city/`](../pilots/covered-city/). For the full research
brief, see the [landing page](../index.html).
